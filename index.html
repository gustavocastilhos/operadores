⚖️ Operadores de Comparação

Eles retornam true ou false.

== Igual

5 == "5" → true

=== Estritamente igual

5 === "5" → false

!= Diferente

10 != 5 → true

> Maior que

10 > 5 → true

< Menor que

2 < 8 → true
🤯 Essa é classicakk:
0 == false → true
javascript as vezes ete o loko! 😂

🧠 Operadores Lógicos

Servem para combinar condições.

&& (AND)

true && true → true

|| (OR)

false || true → true

! (NOT)

!true → false
😂 Exemplo da vida real:

temDinheiro && lojaAberta

Resultado:
🛒 felicidade = true

🔥 Operadores Extras

Incremento

numero++

Decremento

numero--

Ternário

idade >= 18 ? "Maior" : "Menor"

Ternário

idade >= 18 ? "Maior" : "Menor"
💀 Ternário economiza linhas... mas depois de 4 ternários seguidos ninguém entende mais nada.

O operador ternário é uma forma curta de escrever um if/else.


Funciona assim:

condição ? valorSeTrue : valorSeFalse

Exemplo:

idade >= 18 ? "Maior de idade" : "Menor de idade"

Se a condição for verdadeira, retorna o primeiro valor.
Se for falsa, retorna o segundo.
