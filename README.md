# ⚡ JS Operators Guide

Guia interativo sobre operadores JavaScript feito com HTML, CSS e JavaScript.

## 🚀 Sobre o projeto

Este projeto foi criado para estudar e ensinar operadores JavaScript de forma visual, moderna e divertida.

O site possui:

- ✅ Operadores aritméticos
- ✅ Operadores lógicos
- ✅ Operadores de comparação
- ✅ Operadores extras
- ✅ Exemplos práticos
- ✅ Explicações interativas
- ✅ Quiz JavaScript
- ✅ Quiz desafio
- ✅ Interface moderna dark mode

---

## 🛠️ Tecnologias usadas

- HTML5
- CSS3
- JavaScript
